# 使い方動画 制作ベース

## ゴール
- **YouTube 60秒版**: 「使ってみて "おっ、ここいいね" となる体験」を伝える
- **X / Instagram 20秒版**: タイムラインで指を止める。3秒で "写真→盤面" のインパクト

## ファイル構成
- `demo-scenario.md` — 撮影で使う盤面・操作の流れ（両動画共通）
- `script-youtube-60s.md` — YouTube本編の秒割り台本
- `script-shortform-20s.md` — X/Insta縦動画の台本
- `production-checklist.md` — 機材・環境・事前準備・チェックリスト
- `thumbnail-and-copy.md` — サムネ案・タイトル・説明文・ハッシュタグ

## 制作の順序（おすすめ）
1. `demo-scenario.md` の局面を実際の盤で作り、iPhoneで数枚撮っておく（**認識テスト済み**の写真を用意）
2. `production-checklist.md` に沿って収録環境をセット
3. スクリーン録画（iOS標準の画面収録）＋外部カメラで手元を別撮り。**1回の収録で両動画に使える**素材を確保
4. `script-youtube-60s.md` を先に編集（本編）→ 良いカットをXショートに転用
5. サムネ・タイトル・説明文は `thumbnail-and-copy.md` に案あり

## 撮影で「おー」を確実に出す3ポイント
1. **写真が盤面に化ける瞬間**（変換ボタンを押した後の1〜2秒の間）を編集で強調
2. **タップして駒を直せる自然さ**（指の動きを俯瞰でも見せる）
3. **ぴよ将棋への一発連携**（アプリが立ち上がる瞬間まで見せる）

この3つを両動画とも軸にしています。
