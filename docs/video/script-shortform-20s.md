# X / Instagram 縦動画 20秒版 台本

**尺**: 20秒（15秒に圧縮も可）
**アスペクト**: 9:16 縦（1080×1920）
**方針**: **音なしで見られる前提**。テキストオーバーレイ多用・テンポ速め
**音**: 軽快なBGMのみ（フリー音源）

## 秒割り

| 時間 | 画面 | 大テロップ（画面中央〜上） | 補足 |
|---|---|---|---|
| **0–3s** | 盤の実物写真 → **勢いよく盤面イラストへ切替** | **「写真1枚が、これ」**（強め） | ★フック。0.5秒で "写真"、1秒後に "盤面出現"、切替はカット/ワイプ |
| **3–7s** | サイト画面: photo → 変換 → 盤出現 | **「タップ1回で変換」** | 実操作の早送り。**変換ボタン→盤出現までを1.5秒**で見せる |
| **7–12s** | done画面: マスをタップ→駒選択 | **「間違いは指で直せる」** | 指の動き重要。パレットが開く演出 |
| **12–16s** | ぴよ将棋ボタン→アプリ起動 | **「そのまま検討へ」** | アプリ画面まで映す |
| **16–20s** | ロゴ／URL／QR | **「shogi.nkkuma.tokyo」** ＋ **「@nkkuma_service」** | 締め。BGMも落として静かに |

## 縦動画で外せない3原則
1. **最初の1秒に "写真→盤面" の切替を必ず入れる**（スクロールする指を止める）
2. テロップは**画面中央上寄せ**（下1/3はUIの親指エリアで見にくい）
3. 音なし前提。**全情報をテロップで完結**させる
