# 収録・制作 チェックリスト

## 事前準備（撮影の前日まで）

### 盤面の準備
- [ ] `demo-scenario.md` の局面を実物の盤・駒で作る
- [ ] **その盤面を実際にサイトで変換して**正しく認識されるか確認（本番で失敗すると詰む）
- [ ] 認識結果を後で修正する用に、**わざと1マスだけ違う駒**を置くパターンも1つ用意（Wow②のカット用）

### 機材
- [ ] スマホ2台（撮影用と操作用）**または** iPhone画面収録＋外部カメラで手元別撮り
- [ ] スマホスタンド or 三脚
- [ ] 柔らかい照明（自然光か拡散リングライト）— 強い直射日光は影が濃くなるのでNG
- [ ] 音声を入れる場合：外付けマイクかピンマイク（内蔵マイクは環境音が乗る）
- [ ] 編集ソフト：iMovie / CapCut / Premiere / DaVinci のいずれか。**縦動画対応があるもの**

### 環境
- [ ] 静かな時間帯（周囲の話し声・生活音が入らない）
- [ ] 盤の位置は逆光にならない配置
- [ ] スマホの通知はオフ（撮影中に通知バナーが出ると台無し）

## 収録本番（1回のセッションで両動画分を撮る）

### iPhone画面収録の設定
- 設定 → コントロールセンター → 「画面収録」を追加
- **音声も録るか**を左長押しで確認（マイクON推奨）
- 撮影開始 → サイトを操作 → 撮影停止

### 撮る順序（1回で完結させる）
1. サイトのトップ（3–5秒静止で撮影）
2. 「📷 撮影する」タップ → カメラ起動 → 事前撮影の写真を使う場合は「🖼 画像を選ぶ」→ ライブラリから選択
3. photo画面（3–5秒静止）
4. 「変換する」タップ → busy → done（**この一連は途中で止めない**）
5. done画面で1マスタップ → パレット → 正しい駒 → 反映（1回でキメる）
6. 手番セグメントの確認・「KIF」選択・コピー → トースト
7. 「🐤 ぴよ将棋」タップ → アプリ起動

### 手元別撮り（音声・雰囲気用）
- 対局中の実物盤面を1〜2カット
- iPhoneで盤を撮る動作を1カット
- あなたの表情（本編用の "おー" のリアクション）

## 編集

### 共通のコツ
- 6 → 7（busy → done）の**盤が現れる瞬間**は0.5秒スロー＋SEを入れると印象UP
- 修正操作(9–10)は**指の動きにフォーカス**したズームカットを1つ挟む
- ぴよ将棋起動(11–12)は**アプリのロゴが見えるところ**まで映す（連携感）

### YouTube本編
- 60秒に収まらなければ、修正操作の秒数を先に削る
- エンドカードにチャンネル登録ボタン（YouTube機能）を貼る

### Xショート
- **書き出し設定：1080×1920 / 30fps / H.264**
- Xは動画に自動キャプションを付けるので、テロップは既に画面内にある想定でOK
- サムネイル自動選定を確認（1秒目がフックになっているか）
