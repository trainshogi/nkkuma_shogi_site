# YouTube本編 60秒版 台本

**尺**: 60秒
**方針**: 実演→リアクション→出口。「おー、ここいいね」の3拍子を実感で見せる
**音**: あなたのナレーション（推奨・ゆるく・作らない）＋薄い BGM

## 秒割り

| 時間 | 画面 | ナレーション（案・自然な言い回しに変えてOK） | 演出メモ |
|---|---|---|---|
| **0–3s** | 実物盤面（俯瞰）→スマホで撮影する手 | 「対局の途中、この局面をあとでちゃんと検討したい」 | フック。ここで **将棋ファンの共感** を取る |
| **3–8s** | サイトを開く→「📷 撮影する」を押す | 「ShogiBan to Kif、URLは shogi.nkkuma.tokyo」 | 短くサービス名だけ言う |
| **8–13s** | 写真がプレビューされる→「変換する」（赤ボタン）タップ | 「写真1枚、これだけ」 | ボタンをタップする瞬間で**間** |
| **13–18s** | busy画面→**盤が現れる** | 「（無言 or "おー"）」 | ★**Wow①**。**BGMを一瞬止めるか音量を落として盤の出現を強調** |
| **18–28s** | 「あ、ここ違うな」→マスをタップ→パレット→駒を選ぶ | 「間違えてても、その場でタップで直せる」 | ★**Wow②**。指の動きをそのまま入れる |
| **28–38s** | 手番セグメント確認→「KIF」→コピー→トースト | 「棋譜としてKIFやSFENでコピー」 | テキストは短く。トーストの一瞬を映す |
| **38–48s** | 「🐤 ぴよ将棋」タップ→アプリ起動→検討開始 | 「そのまま、ぴよ将棋やKENTOで検討へ」 | ★**Wow③**。アプリ画面まで見せる |
| **48–55s** | サイトのトップに戻る／笑顔などのB-roll | 「盤を、棋譜に。無料・登録不要。」 | 静かに締める |
| **55–60s** | エンドカード：URL / QR / X/フォロー | （テロップのみ） | shogi.nkkuma.tokyo と @nkkuma_service |

## 「60秒で足りるか？」への答え
足ります。ただし条件が2つ：
1. **8–13s の "写真1枚、これだけ" と 13–18s の "変換の瞬間" にちゃんと間を作る**（早口で埋めない）
2. **修正操作(18–28s) は1マスだけ**にする（複数マス直すと尺が破綻）

もし収まりきらない・もっと機能を見せたい場合は**90秒版**にする案：38–48s の後に **「検討して指した次の一手も、また撮って続きから」** のループ体験を10秒追加。

## テロップ（英語圏にも当てたい場合）
- "Photo → Board" (13–18s)
- "Tap to fix" (18–28s)
- "Copy KIF / SFEN" (28–38s)
- "Open in Piyo Shogi / KENTO" (38–48s)
